# AI Smart Traffic Analyzer

An AI-based Smart Traffic Analyzer using YOLOv11 for vehicle detection, counting, and traffic density analysis.

## Features
- Vehicle Detection
- Vehicle Counting
- Traffic Density Estimation
- Traffic Signal Recommendation
- Report Generation

## Technologies
- Python
- YOLOv11
- OpenCV
- Ultralytics
- NumPy
- Pandas
- Matplotlib

## Project Structure

AI-Smart-Traffic-Analyzer/
│
├── models/
├── src/
├── reports/
├── output/
├── requirements.txt
└── README.md


## Installation

```bash
pip install -r requirements.txt
```

## Run

```bash
python src/main.py
```
